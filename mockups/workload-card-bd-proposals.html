🧠 데이터 출처 (구현 시 활용)

  • B (팀원 부하): 기존 teamFeed[].items[]를 멤버별로 그룹핑 — 추가 데이터 없음
  • D (KPI 분포): 같은 task의 task.kpi || task.impact 필드 카운트 — 추가 데이터 없음
  • 쏠림 임계: 1위 KPI가 60%+ 차지 시 황색 배지 (변경 가능)
  • 과부하 임계: 부서 평균 대비 1.5배 이상 / 또는 막대 100% 차지하는 사람 = 빨강
  • 활성 vs 완료 토글: 펼침 카테고리에 따라 같은 정보로 다른 분석 가능 (확장 옵션)
